## 修正内容

### 1. placement_strategy.py — 終盤ダマテン修正（オーラス4位例外追加）

**問題**: `should_damaten()` の巡目 >= 14 ロジックで、オーラス4位もダマテンを推奨していた。
オーラス4位は1000点のリーチ棒コストは無意味（既にラス）で、+1翻のツモ和了確率向上が重要。

**修正前**:
```python
if my_turn >= 14 and not (gs.is_all_last and gs.my_placement == 1):
    return True
```

**修正後**:
```python
if my_turn >= 14 and not (gs.is_all_last and gs.my_placement in (1, 4)):
    return True
```

### 2. game_state.py — 東風戦自動検出の無効化 + set_tonpu() API追加

**問題**: E4到達時の `_round_count >= 4` ヒューリスティックが不信頼。
半荘で親連荘すると E4 に `_round_count >= 4` で到達し、誤って東風戦と判定される。

**修正**:
- 自動検出ヒューリスティックを削除
- `set_tonpu(bool)` メソッドを追加（ロビー情報から明示的に設定）
- S風確認による自動修正（hanchan検出）は維持

## 残課題

1. **手替わり考慮不足**: MAWASHI時の手替わり（向聴数改善の有効牌）を考慮していない
2. **北抜き戦略**: 3人麻雀での北（抜きドラ）の扱いが未実装
3. **染め手判断の高度化**: 自家の染め手進行時の副露・打牌調整が弱い

5. **東風戦検出**: `set_tonpu()` APIは追加したが、ロビー情報との連携が必要
