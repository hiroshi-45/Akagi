# akagi_supreme 評価レポート

## 初回評価: トッププレイヤーの思考を正しく反映できていない → 修正済み

---

## 修正内容まとめ

### 1. game_state.py — 脅威度推定の大幅強化
- **手出し/ツモ切りパターン追跡**: ツモ切り連続後の手出し = テンパイ気配を検出
- **河の染め偏り分析**: ホンイツ/チンイツ/トイトイ気配を副露＋河から検出
- **受入枚数推定**: 手牌構成から有効牌の枚数を概算
- **巡目対応脅威度**: `apparent_threat_level(current_turn)` で巡目を反映
- **赤ドラ二重カウント修正**: `tile_base()` を使い正規化後にドラ照合
- **点差逆転計算**: `points_needed_for_placement()`, `min_han_for_points()` 追加
- **自風/場風/役牌判定**: `is_my_yakuhai()` 追加

### 2. push_fold.py — 巡目ベースの攻守判断
- **巡目 (`my_turn`) を主軸に**: `remaining_tiles` だけでなくプレイヤー巡目で分岐
- **テンパイ閾値厳格化**: トップ勢と同じくテンパイからほぼ降りない（safety_weight極小化）
- **イーシャンテンの適正化**: 序盤/中盤/終盤で異なる safety_weight（0.03〜0.18）
- **打点評価の精密化**: 役牌(自風/場風含む)、ホンイツ、メンゼン立直＋裏ドラ期待を反映
- **打点→翻数変換**: 正しい点数表ベースの翻数→点数マッピング
- **脅威度に行動パターン反映**: `tedashi_after_tsumogiri_streak()` によるダマ聴検出

### 3. placement_strategy.py — 精密なオーラス戦略
- **逆転条件計算**: `points_needed_for_placement()` + `min_han_for_points()` でツモ/ロン別に必要翻数を算出
- **ダマテンの適用範囲拡大**: オーラス1位だけでなく、南場1位・跳満以上の手でも適用
- **供託棒の戦略反映**: kyotaku_bonus を all-last, south の両方で参照
- **親番4位連荘戦略**: 副露推奨 (meld_multiplier=1.2) で速攻アガリ優先
- **2位/3位の精密分岐**: 1位との翻数差、3位との点差を同時考慮

### 4. strategy_engine.py — バグ修正と判断改善
- **HORA 見逃しバグ修正**: `mask[IDX_HORA]` チェックを `mortal_action` チェックより前に移動
- **デッドパラメータ削除**: `_find_best_discard()` から未使用の `safety_weight` 引数を除去
- **Q値正規化改善**: min-max → パーセンタイルベース (10th/90th) で外れ値に耐性
- **danger 正規化改善**: リーチ有無で max_danger を 1.8/1.2 に動的切替
- **副露判断の強化**: FOLD 時のポン判断、CAUTIOUS 時のチー判断にQ値差分を考慮
- **リーチ判断の閾値修正**: 0.85 → 0.90 でMortalのリーチ判断をより尊重

## 設計方針の変更

**修正前**: 多くの場面でルールベースでMortalを上書き
**修正後**: オーバーライドを「順位戦略が明確に必要な場面」に限定し、Mortalを最大限信頼

- safety_weight の全体的な引き下げ（トップ勢は攻撃的）
- テンパイ/イーシャンテンでのMortal信頼度を大幅向上
- オーラス・南場の順位条件のみで積極的に介入
